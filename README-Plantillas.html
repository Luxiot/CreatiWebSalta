Localiza la sección `:root` en el CSS 2. Modifica los valores hexadecimales 3. Los cambios se aplicarán automáticamente en toda la plantilla --- ## 📱 **Secciones Incluidas** ### **Todas las Plantillas Incluyen:** - ✅ Header con navegación fija - ✅ Hero section impactante - ✅ Secciones de contenido principales -
